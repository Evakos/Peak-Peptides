# Peak Peptides Landing Page

Elite peptide performance protocol landing page with dark boutique design.

## Features

- **Dark Boutique Design**: Black/gray theme with red accents and grayscale imagery
- **Sales Conversion Elements**: Pricing tiers, countdown timer, testimonials, FAQ, guarantee section
- **Modern Stack**: React, TypeScript, Tailwind CSS v4
- **Custom Typography**: Cal Sans font
- **Responsive Design**: Mobile-first approach
- **Performance Optimized**: Built with Vite

## Tech Stack

- React 18.3.1
- TypeScript
- Tailwind CSS v4
- Vite 6.3.5
- Lucide React (icons)
- Motion (animations)

## Getting Started

### Prerequisites

- Node.js 18+ 
- pnpm

### Installation

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Build for production
pnpm build
```
